# Color Shooting

## ***SEE DEMO FIRST and tell me HOW YOU LIKE MY MUSIC!!!***

## how to compile

### by IDE supporting CMake

Directly do it by IDE. e.g. Clion

### by command line

Run:

```
$ cmake CMakelist.txt
$ make
$ ./ColorShooting
```

CMake is needed.
